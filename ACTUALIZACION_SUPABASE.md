# Nota histórica (Supabase)

Este archivo describía credenciales de un proyecto **Supabase** antiguo. **LikeNew en producción usa PostgreSQL gestionado en DigitalOcean**, no Supabase.

Para desplegar y configurar `DATABASE_URL` hoy, sigue **[DEPLOY_DIGITALOCEAN.md](DEPLOY_DIGITALOCEAN.md)** y **[BASE_DE_DATOS.md](BASE_DE_DATOS.md)**.

Si en el futuro usaras Supabase u otro proveedor, basta con una `DATABASE_URL` `postgresql://...` válida para ese servicio.
