# Project Summary
LikeNew is a web application that offers a professional trash can cleaning service tailored for residents of Las Vegas, Nevada. Users can easily schedule same-day appointments that align with their trash collection days. The platform features a user-friendly management system, booking history, and in-app notifications to enhance user experience. Recent updates have improved in-app notifications and introduced loyalty discounts for returning customers, eliminating external communication methods.

# Project Module Description
- **Frontend**: Contains user interface components, pages, and styling.
- **Backend**: Manages data interactions, authentication, and business logic.
- **Database Models**: Structures for appointments, schedules, and days off.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy
- **Database**: PostgreSQL
- **Dependency Management**: pnpm

# Usage
To install dependencies, navigate to the backend and frontend directories and run:
```
pnpm install
```
To build the application, run:
```
pnpm run build
```
To start the application, execute the main backend script.
